# 自在东方 - 虚实联动的沉浸式互动剧本杀乐园

## 项目简介

【自在东方】是基于上海历史发展的文明体验剧本杀小程序高保真原型。项目包含三个文明时期：
- **崧泽·良渚文明（源点）** - 远古纪元
- **唐宋·青龙镇（主轴）** - 海丝盛景  
- **近代·开埠至今（汇流）** - 洋场新韵

## 项目结构

```
自在东方2/
├── app.html                # 单页面应用主入口（推荐使用）
├── app.js                  # 单页面应用路由系统
├── index.html              # 原型展示页面，使用iframe嵌套展示所有原型
├── home.html               # 主页（购票、角色测试、我的剧本、360全景等）
├── world.html              # 世界页面（线下园区导航、数字孪生）
├── social.html             # 社交页面（万象界、配对、扫码、圈子）
├── profile.html            # 个人中心（数字藏馆、我的动态、订单等）
├── character-test.html     # 角色测试页面
├── my-script.html          # 我的剧本页面
├── styles/
│   └── main.css            # 主样式文件（包含设计规范）
└── README.md               # 项目说明文档
```

## 设计规范

### 色彩体系
- **主色调**：故宫红 (#C8102E)
- **辅助色**：黛青 (#2F4F4F)、赭石色 (#8B4513)
- **背景色**：米白 (#F5F5DC)、浅棕 (#D2B48C)

### 字体体系
- **功能性文字**：无衬线字体（PingFang SC, Helvetica Neue）
- **品牌标题**：书法字体（KaiTi, STKaiti）

### 间距体系
- 采用8px网格系统
- 基础间距：8px, 16px, 24px, 32px, 48px

### 布局规范
- iPhone全面屏适配（375x812px）
- 安全边距：24px（左右）
- 底部导航栏高度：80px

## 技术栈

- **HTML5** - 页面结构
- **Tailwind CSS** - 样式框架
- **FontAwesome** - 图标库
- **原生 JavaScript** - 交互功能

## 使用说明

### 方式一：单页面应用（推荐）
1. 直接在浏览器中打开 `app.html` 查看完整的交互式应用
2. 支持页面间跳转、下拉刷新等功能
3. 底部导航栏可以切换不同页面
4. 所有页面整合在一个应用中，体验更流畅

### 方式二：原型展示
1. 直接在浏览器中打开 `index.html` 查看所有原型页面
2. 以网格形式展示所有功能页面，方便对比和展示
3. 点击"打开完整应用"按钮可跳转到单页面应用

### 方式三：独立页面
1. 各个功能页面可以独立打开查看（home.html、world.html等）
2. 所有页面都模拟了iPhone全面屏的外观
3. 适合单独测试某个功能模块

## 图片资源

- 所有图片均使用 Unsplash 的占位图片

- 实际开发时可以替换为真实图片
- 地图、背景图等位置已标注，可插入实际图片

## 转换为Figma

所有页面使用标准的HTML+CSS结构，可以：
1. 使用Figma的导入功能导入HTML
2. 或者使用在线工具将HTML转换为Figma设计稿
3. 保持Tailwind CSS类名便于后续开发
